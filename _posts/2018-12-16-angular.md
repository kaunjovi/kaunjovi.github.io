## Dependency injection of Service in a Component

In Component

```
constructor(){
  let service = new MyService() ; 
  this.data = service.getData() ;   
}
```

- Or you could have angular created the service instance. 

```javascript 
constructor (service : Service ){
  this.data = service.getData() ; 
}
```
